# アドフリー・オートクリッカー (Ad-Free Auto Clicker)

このアプリは、Android 7.0 (API 24) 以上で動作する、広告なし・タスク数無制限のオートクリッカーです。

## 特徴
- **広告なし**: 完全に無料で、広告表示は一切ありません。
- **タスク数無制限**: 作成したクリックパターンをいくつでも保存・管理できます。
- **マルチターゲット対応**: 複数の地点を順番にクリックしたり、スワイプしたりすることが可能です。
- **オーバーレイUI**: 他のアプリの上で操作を開始・停止できる便利なフローティングパネルを搭載。

## 使い方
1. アプリを起動し、「アクセシビリティサービスを有効にする」ボタンをタップします。
2. Androidの設定画面が開くので、「オートクリッカー・サービス」を探してオンにします。
3. アプリに戻り、「新規タスク作成」をタップしてフローティングパネルを表示させます。
4. 他のアプリ（ゲームなど）を開き、パネルの「+」ボタンでクリック地点を追加します。
5. 「▶」ボタンを押すと自動クリックが開始されます。

## ビルド方法
このプロジェクトは Android Studio (Kotlin) で開発されています。

1. Android Studio を開き、`File > New > Import Project` からこのディレクトリを選択します。
2. 必要な依存関係（GSON, AppCompat, RecyclerView等）が自動的にダウンロードされます。
3. `Build > Build Bundle(s) / APK(s) > Build APK(s)` を選択してAPKを生成します。

## 注意事項
- このアプリは Android の `AccessibilityService` を使用してクリックを代行します。
- セキュリティ上の理由から、信頼できる環境でのみ使用してください。
